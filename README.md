### 使用

1. 安装docker
```
```

2. 克隆项目

```
$ git clone git@github.com:WencoChen/dnmp.git
```

3. 个性化配置

```
$ cd dnmp
$ cp docker-compose.yml.example docker-compose.yml
$ cp env.example .env
```

4. 运行

```
$ docker-compose up -d
```
